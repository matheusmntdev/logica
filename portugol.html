Portugol

Portugol é uma pseudolinguagem didática usada para ensinar lógica de programação, com comandos em português que facilitam o aprendizado antes de migrar para linguagens como C, Java ou Python. Este guia aborda sua estrutura, sintaxe, tipos de dados, comandos, estruturas de controle e exemplos práticos, com base em ferramentas como VisuAlg e Portugol Studio.

1. Introdução ao Portugol

Portugol não é uma linguagem de programação compilada, mas um pseudocódigo voltado para o ensino. Ele permite que iniciantes compreendam conceitos de programação sem se preocupar com a complexidade de sintaxes de linguagens reais. É amplamente usado em ferramentas como:

  • VisuAlg: Um interpretador gráfico simples e popular.
  • Portugol Studio: Ambiente moderno com destaque de sintaxe e depuração.
  • Ambientes acadêmicos: Usado para planejar algoritmos em pseudocódigo.

A sintaxe pode variar entre ferramentas, mas os conceitos fundamentais permanecem consistentes.

2. Estrutura Básica de um Programa

Um programa em Portugol segue uma estrutura padrão, com a definição do programa e uma função principal chamada inicio().

Sintaxe Básica

programa {
    // Declaração de variáveis
    // Funções (se houver)
    funcao inicio() {
        // Código principal
    }
}
  • programa: Define o início do programa.
  • funcao inicio(): Ponto de entrada onde o código principal é executado.
  • //: Comentários de uma linha.
  • /* */: Comentários de múltiplas linhas.

3. Tipos de Dados

Portugol suporta tipos de dados básicos para armazenar diferentes tipos de valores.

  • inteiro: Números inteiros (ex.: 5, -10).
  • real: Números com ponto flutuante (ex.: 3.14, -0.5).
  • caractere: Cadeias de texto ou caracteres (ex.: "Olá", 'A').
  • logico: Valores booleanos (verdadeiro ou falso).
  • vetor ou matriz: Estruturas para armazenar múltiplos valores.

Exemplo de Declaração

inteiro idade
real salario
caractere nome[50]
logico ativo
inteiro notas[5] // Vetor de 5 posições

4. Entrada e Saída de Dados

Portugol oferece comandos simples para interagir com o usuário.

Saída: Comando escreva

Exibe informações na tela.

escreva("Olá, mundo!") // Exibe texto
escreva(idade)         // Exibe o valor de uma variável

Entrada: Comando leia

Lê dados fornecidos pelo usuário.

leia(idade) // Lê um valor inteiro
leia(nome)  // Lê uma string

Exemplo Completo

programa {
    funcao inicio() {
        inteiro idade
        escreva("Digite sua idade: ")
        leia(idade)
        escreva("Sua idade é: ", idade)
    }
}

5. Operadores

5.1. Operadores Aritméticos

  • +: Soma
  • -: Subtração
  • *: Multiplicação
  • /: Divisão
  • div: Divisão inteira (retorna a parte inteira)
  • mod: Módulo (resto da divisão)
  • ^: Exponenciação (depende da ferramenta)
inteiro a = 10, b = 3
escreva(a + b)  // Resultado: 13
escreva(a div b) // Resultado: 3
escreva(a mod b) // Resultado: 1

5.2. Operadores Relacionais

  • >: Maior que
  • <: Menor que
  • >=: Maior ou igual
  • <=: Menor ou igual
  • =: Igualdade
  • <>: Diferente

5.3. Operadores Lógicos

  • e: AND (e lógico)
  • ou: OR (ou lógico)
  • nao: NOT (negação)
logico resultado = (5 > 3) e (2 < 4) // Verdadeiro
escreva(resultado)

6. Estruturas de Controle

6.1. Condicionais

As estruturas condicionais permitem executar blocos de código com base em condições.

Comando se e senao

programa {
    funcao inicio() {
        inteiro idade
        escreva("Digite sua idade: ")
        leia(idade)
        se (idade >= 18) {
            escreva("Maior de idade")
        } senao {
            escreva("Menor de idade")
        }
    }
}

Comando escolha (switch/case)

programa {
    funcao inicio() {
        inteiro dia
        escreva("Digite um número de 1 a 7: ")
        leia(dia)
        escolha (dia) {
            caso 1:
                escreva("Domingo")
                pare
            caso 2:
                escreva("Segunda")
                pare
            caso contrario:
                escreva("Dia inválido")
        }
    }
}

6.2. Laços de Repetição

Laço enquanto (while)

Repete enquanto a condição for verdadeira.

inteiro i = 1
enquanto (i <= 5) {
    escreva(i, " ")
    i = i + 1
}
// Saída: 1 2 3 4 5

Laço para (for)

Usa uma variável de controle para repetições.

para (inteiro i = 1; i <= 5; i++) {
    escreva(i, " ")
}
// Saída: 1 2 3 4 5

Laço faca-enquanto (do-while)

Executa pelo menos uma vez e repete enquanto a condição for verdadeira.

inteiro i = 1
faca {
    escreva(i, " ")
    i = i + 1
} enquanto (i <= 5)
// Saída: 1 2 3 4 5

7. Vetores e Matrizes

7.1. Vetores

Arrays unidimensionais para armazenar múltiplos valores.

inteiro notas[5]
notas[0] = 10
notas[1] = 8
escreva(notas[0]) // Saída: 10

7.2. Matrizes

Arrays bidimensionais.

inteiro matriz[2][2]
matriz[0][0] = 1
matriz[0][1] = 2
matriz[1][0] = 3
matriz[1][1] = 4
escreva(matriz[0][0]) // Saída: 1

7.3. Percorrendo Vetores e Matrizes

para (inteiro i = 0; i < 2; i++) {
    para (inteiro j = 0; j < 2; j++) {
        escreva(matriz[i][j], " ")
    }
}
// Saída: 1 2 3 4

8. Funções

Funções permitem modularizar o código, tornando-o reutilizável.

Exemplo com Retorno

funcao inteiro soma(inteiro a, inteiro b) {
    retorne a + b
}

funcao inicio() {
    inteiro resultado = soma(5, 3)
    escreva("Soma: ", resultado) // Saída: 8
}

Exemplo sem Retorno

funcao exibirMensagem(caractere msg[]) {
    escreva(msg)
}

funcao inicio() {
    exibirMensagem("Bem-vindo!") // Saída: Bem-vindo!
}

9. Boas Práticas

  1. Use nomes descritivos: Prefira idade ou salario a x ou y.
  2. Comente o código: Explique trechos complexos para facilitar a leitura.
  3. Indentação: Mantenha o código organizado com recuos consistentes.
  4. Teste incrementalmente: Execute o programa em etapas para identificar erros.
  5. Evite redundância: Use funções para reutilizar lógica.

10. Exemplo Completo

Programa que calcula a média de 3 notas e verifica se o aluno foi aprovado:

programa {
    funcao inicio() {
        real notas[3], soma = 0.0, media
        inteiro i
        
        // Leitura das notas
        para (i = 0; i < 3; i++) {
            escreva("Digite a nota ", i+1, ": ")
            leia(notas[i])
            soma = soma + notas[i]
        }
        
        // Cálculo da média
        media = soma / 3
        
        // Exibição do resultado
        escreva("Média: ", media)
        se (media >= 7) {
            escreva("\nAprovado!")
        } senao {
            escreva("\nReprovado!")
        }
    }
}

11. Ferramentas para Portugol

  • VisuAlg: Interface simples, ideal para iniciantes. Disponível para download gratuito.
  • Portugol Studio: Moderno, com destaque de sintaxe e depuração. Também gratuito.
  • Editores de texto: Podem ser usados para escrever pseudocódigo, mas sem execução automática.

12. Diferenças Regionais e Ferramentas

A sintaxe do Portugol pode variar entre ferramentas:

  • VisuAlg: Usa escreva, leia, se, senao.
  • Portugol Studio: Pode usar escreval (escreve e pula linha) ou sintaxes próximas de C.

Consulte a documentação da ferramenta para confirmar a sintaxe.

13. Limitações do Portugol

  • Não compilado: Não gera executáveis para uso em produção.
  • Focado em ensino: Não suporta recursos avançados como manipulação de arquivos ou redes.
  • Variações de sintaxe: Podem causar confusão ao mudar de ferramenta.

14. Transição para Outras Linguagens

Portugol prepara o aluno para linguagens como:

  • C: Estruturas como for, while e if são semelhantes.
  • Python: Simplicidade na sintaxe, mas com mais recursos.
  • Java: Mais robusto, com conceitos de orientação a objetos.

Exemplo de Equivalência

// Portugol
escreva("Olá")

// C
printf("Olá\n");
